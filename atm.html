About the ATM Program

6th Grade

In 6th grade, we learned typing skills, cyber safety, how to use Microsoft apps, such as Word, PowerPoint, and Excel, and how to use Google apps as well, such as Docs, Slides, Sheets, and Drawings.

7th Grade

In 7th grade, we learned how to use Adobe apps such as PhotoShop, Illustator, and InDesign. We also learned how to design in 3D using a program called SketchUp.

8th Grade

So far in 8th grade, we have learned how to code webpages using a program called Visual Studio Code and websites called Dash and Khan Academy. Next, we will learn how to film using different types of equipment, how to edit videos in iMovie, and how to add sound effects in Garage Band.

ATM Reflection

The ATM program is really amazing and it’s extremely helpful to students who are just barely being introduced to a relatively new computer and technology. It’s been especially helpful to me because I’ve been able to discover new programs and methods of doing things that I’ve never been able to do before. I truly believe that because of this ATM program, my future, such as high school or working in a job, will be a lot easier than it has ever been before. I think that anyone who enjoys using technology will just love the ATM program!
