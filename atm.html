About the ATM Program

6th Grade

In 6th grade, we learned typing skills, cyber safety, how to use Microsoft apps, such as Word, PowerPoint, and Excel, and how to use Google apps as well, such as Docs, Slides, and Sheets.

7th Grade

In 7th grade, we learned how to use Adobe apps such as PhotoShop, Illustrator, and InDesign. We also learned how to design in 3D using a program called SketchUp.

8th Grade

So far in 8th grade, we have learned how to code web pages using a program called Visual Studio Code and websites called Dash and Khan Academy. Afterwards, we will learn how to film, how to edit videos in iMovie, and how to add sound effects in Garage Band.

Ilya Pavlov

ATM Curriculum

This is what you will learn (or at least what I learned, it might differ based on teachers and what year that you're in the class) while in the ATM program.

6th Grade 7th Grade 8th Grade
How to type properly and improve wpm (words per minute) How to use PhotoShop to edit and alter photos How to use Visual Studio Code to create websites
How to be safe on the internet How to use Illustrator to create logos How to use Dash to create a website and a blog
How to use Microsoft Apps, such as Word, PowerPoint, and Excel How to use InDesign to create magazine covers How to use iMovie to film and edit movies
How to use Google Apps, such as Docs, Slides, and Sheets How to design objects and buildings in 3D using SketchUp How to use Garage Band to add sound effects to a video

Teens Dream Competition

In my 8th grade year, my teacher had us make a video for the Teens Dream Competition.

Audio Production Unit

The Audio Production Unit was really cool. It allowed us to work in GarageBand and makes our own songs. A big project that we had was when we were handed a silent animation, tasked with the job of adding in sound effects.

ATM Reflection

The ATM program is really amazing and it’s extremely helpful to students who are just barely being introduced to a relatively new computer and technology. It’s been especially helpful to me because I’ve been able to discover new programs and methods of doing things that I’ve never been able to do before. I truly believe that because of this ATM program, my future, such as high school or working in a job, will be a lot easier than it has ever been before. I think that anyone who enjoys using technology will just love ATM!

Gabriel Beaudry
